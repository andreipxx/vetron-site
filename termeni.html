2. Descrierea serviciului

VETRON este o aplicatie Android care analizeaza ofertele de curse Bolt in timp real, calculand profitabilitatea pe baza costurilor reale ale utilizatorului (carburant, uzura vehicul, taxe).

Aplicatia afiseaza un verdict vizual prin intermediul unui overlay pe ecranul telefonului:

  • GO — cursa este profitabila conform parametrilor tai
  • THINK — profitabilitatea este marginala, analizeaza inainte de decizie
  • STOP — cursa nu este profitabila conform parametrilor tai

VETRON este un instrument de analiza financiara. Aplicatia nu interactioneaza cu aplicatia Bolt, nu accepta si nu refuza curse in locul soferului si nu transmite date catre platforma Bolt.
